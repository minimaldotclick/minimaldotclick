Digital Tools

I'm trying to make sure that I use only open source tools when I create and build. Here is an non-exhaustive list of them:

**Zim**: What I use to build my wiki (and this website).

HTML/CSS: Development tools for my website.

Python: a high-level, interpreted, general-purpose programming language.

Ruby: an interpreted, high-level, general-purpose programming language which supports multiple programming paradigms.

Godot: a really simple game engine suitable for designing games and apps across a variety of platforms. Both 2D and 3D.

Blender: 3D an 2D modelling animation and rendering suite. Excellent for game development, making animated movies, concept art, VFX, practically anything.

Krita: painting software. All the tools you need to grow as an artist.

Inkscape: vector art software.

Ardour: a digital audio workstation for creating professional music.

LMMS: a digital audio workstation for creating professional music.

Piskel: pixel art and animated sprite editor.

Firefox: the only privacy focussed browser there is.

Calibre: ebook management.

There is also a host of online resources that I use to help facilitate the running of minimal.click
